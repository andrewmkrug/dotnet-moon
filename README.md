# dotnet moon repo - monorepo

## Setup 

1. Clone this repo
1. Install [proto](https://moonrepo.dev/docs/proto/install). 
1. Install all tools

```sh
proto use
```

### Mac Setup on zsh

``` sh
brew install proto
source ~/.zshrc
proto use
```

## General Use

| Task | Command |
| --- | --- |
| Run Affected Targets | `moon ci` |
